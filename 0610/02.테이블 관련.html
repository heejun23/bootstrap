table관련 class 서식

  • class="table" : 테이블 기본서식이 적용됨
  • class="table-striped" : 행마다 다른 색상 적용
  • class="table-bordered" : 테이블 셀 테두리 만들기
  • class="table-hover" : 마우스 오버시 배경색 변경
  • class="table-dark" : 테이블 제목줄에 배경색 어둡게 적용
  • class="table-sucess / table-danger / table-warning / table-info / table-primary등 색상태그" : tr,td에 각 분류색상 적용가능
  • class="table-responsive" : 화면너비가 768px미만일 경우 가로스크롤됨
