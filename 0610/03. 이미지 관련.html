이미지관련

이미지태그를 사용해 이미지를 삽입하고 다양하게 클래스를 적용해본다.

1. 모서리를 둥글게 css(border-radius)

부트스트랩에서는 class="rounded"로 모서리 둥글게 적용


2. 모서리를 더 둥글게 해서 원이미지로 만들기 class="rounded-circle"

3. 이미지를 갤러리처럼 썸네일 이미지로 보이게 class="img-thumbnail"

border, padding, border-radius 모두 적용

4. 이미지와 문자를 사용해 콘텐츠 제작시 figure, figcaption 사용할 경우

이미지를 반응형으로 동작하는 클래스 class="img-fluid" 적용시 width=100% height=100% 자동으로 들감

부몬의 크기가 변하면 자식도 같이 변함
